# 欢迎使用 Cloudflare Workers Blog

这是一个使用 Cloudflare Workers 和 KV 存储构建的博客系统。

## 特性

- ✅ 基于 Cloudflare Workers 运行
- ✅ 使用 KV 存储文章数据
- ✅ 支持 Markdown 编辑
- ✅ 自动生成静态页面
- ✅ 支持分类和标签
- ✅ 响应式设计

## 使用方法

1. 在本地编写 Markdown 文件
2. 使用脚本转换为 JSON 格式
3. 自动部署到 Cloudflare Workers
4. 文章会自动存储到 KV 中

## 总结

这个博客系统提供了一个简单而强大的方式来管理和发布您的文章。
